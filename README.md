Weather & Emergency Response Dashboard
An interactive web application that combines real-time weather alerts, earthquake data, and instant emergency video collaboration using Pexip into a single operational dashboard.
This tool is designed for situational awareness and rapid response workflows, providing a map-first interface that allows users to monitor critical events and immediately launch a video call into a predefined emergency meeting room.

🚀 Features
🌦 Weather Intelligence (NWS)

Live weather alerts from the National Weather Service
Filterable by:

Region (Continent → Country → Region)
Event type (e.g., tornado, flood, storm)
Severity


Map-based visualization with clickable event markers


🌍 Earthquake Monitoring (USGS)

Real-time earthquake data from USGS
Configurable filters:

Magnitude
Time window
Location radius


Displayed both on the map and in list form


🗺 Interactive Map

Map-first UI (Leaflet)
Clickable markers for:

Weather events
Earthquakes


Dynamic map behavior:

Auto-zoom to region selection
Region-based filtering


Flexible geographic hierarchy:

Continent → Country → Administrative Region




📍 Region-Based Filtering

Global-ready geographic filtering system
Supports:

Broad views (continent-level)
Focused views (state/province level)


Optional selection depth:

You can filter at any level (no forced drill-down)




📞 Emergency Calling (Pexip)

One-click launch into a Pexip Virtual Meeting Room (VMR)
Uses Web App 3 join link model
